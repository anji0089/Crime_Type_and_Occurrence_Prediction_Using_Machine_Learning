# Crime Type and Occurrence Prediction Using Machine Learning

## Overview

This project focuses on predicting crime types and crime occurrence patterns using machine learning techniques. The system analyzes historical crime data and identifies patterns based on features such as location, time, crime category, and other related factors.

The main goal of this project is to support crime analysis and prediction by using data-driven methods. It can help in understanding crime trends and identifying areas or conditions where certain types of crimes may occur more frequently.

## Objective

- To analyze historical crime data using machine learning.
- To predict the type of crime based on available data features.
- To identify crime occurrence patterns based on time and location.
- To apply data preprocessing, feature selection, and model training.
- To evaluate machine learning models using performance metrics.

## Key Features

- Crime data preprocessing and cleaning
- Exploratory data analysis
- Feature selection and transformation
- Machine learning-based crime type prediction
- Crime occurrence pattern analysis
- Model training and evaluation
- Accuracy and performance comparison

## Project Workflow

1. **Dataset Collection**
   - Used historical crime-related data for analysis and prediction.

2. **Data Preprocessing**
   - Removed missing values and duplicate records.
   - Cleaned and formatted the dataset.
   - Converted categorical values into numerical format where required.

3. **Exploratory Data Analysis**
   - Analyzed crime trends based on available dataset features.
   - Studied patterns related to crime type, time, and location.

4. **Feature Selection**
   - Selected important features that help in predicting crime type and occurrence.

5. **Model Training**
   - Trained machine learning models using the processed dataset.

6. **Model Evaluation**
   - Evaluated model performance using accuracy and other classification metrics.
